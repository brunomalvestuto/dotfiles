# DOTFILES

## Requirements

* zsh
* vim
* oh-my-zsh
* tmux


## Install

Fire up the commands from within your home folder

    git clone https://github.com/brunomalvestuto/dotfiles &&
    cd dotfiles
    ./install.sh
